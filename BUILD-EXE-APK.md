# Сборка настоящих EXE и APK (PWABuilder)

После публикации онлайн (см. `DEPLOY-GITHUB.md`) можно собрать полноценные установщики.
Инструмент: **https://www.pwabuilder.com** (бесплатный, от Microsoft). Ничего ставить не нужно.

Оба пакета **сами обновляются** из онлайн-версии — пересобирать при каждом обновлении не надо.

## Подготовка
1. Открой **https://www.pwabuilder.com**.
2. Вставь адрес своей онлайн-версии (напр. `https://твой-логин.github.io/job-helper/`) → **Start**.
3. Он проверит PWA. Если попросит иконки — нажми **Generate**/*Generate Images* и загрузи `icons/icon.svg`
   (или любой квадратный PNG 512×512). PWABuilder сам сделает все размеры.

## 📱 Android (APK) — для Xiaomi 14 Ultra
1. На странице результатов выбери **Android → Generate Package**.
2. Пакет: **Google Play** оставь по умолчанию, но для установки «мимо магазина» тебе нужен **APK**:
   - В окне генерации включи/убедись, что выдаётся **APK** (а не только AAB).
   - **Signing key:** выбери **Create new** (PWABuilder создаст ключ и положит его в архив — **сохрани этот архив**, он нужен для будущих обновлений подписи).
3. Скачается ZIP. Внутри — файл **`.apk`** и данные ключа.
4. Перекинь `.apk` на телефон (кабель/облако), открой его на телефоне, разреши «установку из неизвестных источников» для проводника/браузера → **Установить**.

> На MIUI/HyperOS (Xiaomi) при установке отключи в настройках безопасности «Проверку приложений»/«MIUI Optimization», если ругается.

## 💻 Windows (EXE/приложение) — для Win11 x64
1. На странице результатов выбери **Windows → Generate Package**.
2. Получишь **MSIX**-пакет (современный установщик приложений Windows) + `.exe`-скрипт установки/тестовый сертификат.
3. Установка:
   - Распакуй ZIP.
   - Кликни правой кнопкой по файлу **`install.ps1`** → *Выполнить с помощью PowerShell* (он поставит тестовый сертификат и приложение).
   - Либо: сначала установи сертификат (`.cer`) в «Доверенные корневые», затем двойной клик по `.msix`.
4. Приложение появится в «Пуске» как обычная программа, в своём окне.

> Если хочешь именно классический `.exe`-установщик без MSIX/сертификатов — скажи, соберём альтернативой на **Tauri** (лёгкий, ~5–10 МБ, один `.exe`). Это отдельный маршрут со сборочными инструментами.

## Самый быстрый путь (без сборки)
Если возиться со сборкой не хочется — просто **установи как PWA**:
- ПК: Edge → открыть адрес → «Установить приложение».
- Телефон: Chrome → открыть адрес → «Установить приложение».
Выглядит и работает как обычное приложение, обновления те же.
