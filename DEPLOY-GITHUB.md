# Публикация онлайн (GitHub Pages)

Онлайн-версия нужна для: установки на телефон, обновлений по сети и сборки exe/apk.
Бесплатно и навсегда.

## Вариант А. Через сайт GitHub (без командной строки) — проще

1. Зарегистрируйся/войди на **https://github.com**.
2. Нажми **+** (вверху справа) → **New repository**.
   - Repository name: `job-helper` (или любое).
   - Public.
   - **Create repository**.
3. На странице репозитория: **Add file → Upload files**.
4. Перетащи в окно **всё содержимое** папки `JobHelper` (файлы `index.html`, `sw.js`, `manifest.webmanifest`, папки `css`, `js`, `icons`).
   - Важно: заливай именно файлы/папки, а не саму папку `JobHelper` целиком, чтобы `index.html` оказался в корне репозитория.
5. Внизу **Commit changes**.
6. Открой **Settings** (репозитория) → слева **Pages**.
   - Source: **Deploy from a branch**.
   - Branch: **main** и папка **/(root)** → **Save**.
7. Через 1–2 минуты вверху появится адрес вида:
   **`https://ТВОЙ-ЛОГИН.github.io/job-helper/`**

Это и есть адрес приложения. Открой его в браузере — всё должно работать, а обновления теперь «живые».

## Вариант Б. Через командную строку (если знаком с git)

```bash
cd "путь/к/JobHelper"
git init
git add .
git commit -m "JobHelper 1.0.0-beta"
git branch -M main
git remote add origin https://github.com/ТВОЙ-ЛОГИН/job-helper.git
git push -u origin main
```
Затем включи Pages как в пунктах 6–7 варианта А.

## Как выкатывать ОБНОВЛЕНИЯ потом
1. Я вношу изменения в файлы и поднимаю версию (в `js/version.js` и `CACHE_VERSION` в `sw.js`).
2. Ты заливаешь изменённые файлы в тот же репозиторий (Upload files / `git push`).
3. На ПК и телефоне приложение само заметит новую версию и покажет кнопку **«Обновить»**.
   Никаких переустановок.

## Установка после публикации
- **ПК (Edge/Chrome):** открой адрес → в адресной строке значок «Установить» (или меню → «Установить приложение»).
- **Телефон (Chrome на Android):** открой адрес → меню (⋮) → **«Установить приложение» / «Добавить на главный экран»**.
- Настоящие **exe/apk** — см. `BUILD-EXE-APK.md`.
